# 🍕 Pizza Sales Report Dashboard

## 📌 Overview
This project is an interactive **Power BI dashboard** designed to analyze pizza sales performance using business intelligence techniques. It provides valuable insights into sales trends, customer preferences, and product performance through interactive visualizations, enabling data-driven decision-making.

## 🎯 Objectives
- Analyze overall sales performance.
- Track revenue, orders, and pizzas sold.
- Identify top and bottom-selling pizzas.
- Analyze sales by category, size, and time.
- Provide actionable business insights.

## 🛠️ Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

## 📊 Dashboard Features
- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Average Pizzas per Order
- Daily & Monthly Sales Trends
- Sales by Pizza Category
- Sales by Pizza Size
- Top 5 Best-Selling Pizzas
- Bottom 5 Selling Pizzas
